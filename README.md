# 🥁 Drum Charts

A drum chart builder with performance mode. Built as a single-page HTML app.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (can be private or public)
2. Upload `index.html` and this `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select **Deploy from a branch**
5. Set branch to `main`, folder to `/ (root)`, click Save
6. Your site will be live at `https://yourusername.github.io/your-repo-name` within ~60 seconds

## Notes

- Songs are saved to your browser's `localStorage` on each device
- Use **Export All** to download a `.json` backup of your library
- Use **Import** to load that backup on a new device or browser
